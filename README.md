# eTuitionBD

**eTuitionBD** is a web-based Tuition Management System where **students**, **tutors**, and **admins** can manage tuition-related activities in one platform.

Students can post tuition requirements, tutors can apply to suitable tuitions, and admins can monitor and control the system to ensure quality and trust.

## What You Can Do

### Students

- Register and log in
- Post tuition requirements
- View and manage posted tuitions
- Review tutor applications
- Approve tutors after payment
- View payment history
- Update profile information

### Tutors

- Register and log in
- Browse approved tuition posts
- Apply for tuitions
- Track application status
- View ongoing tuitions
- Check earnings history

### Admins

- Manage users (students, tutors, admins)
- Approve or reject tuition posts
- Monitor applications
- View platform transactions and reports

---

## Main Features

- Role-based authentication (Student / Tutor / Admin)
- Secure login with Firebase & JWT
- Stripe payment integration
- Responsive dashboard layouts
- Search, filter, and pagination for tuition listings
- Clean and user-friendly UI
